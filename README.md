# AI Agents Workspace

This workspace is for testing AI-agentic design, AI composition workflows, and small game experiments.

## Structure

- `game-ai/` - current browser game project.
- `docs/` - workflow and architecture documentation.
- `experiments/` - future prototypes and isolated tests.
- `CODEX.md` - Codex behavior and handoff rules.
- `CLAUDE.md` - Claude behavior and planning rules.
- `GEMINI.md` - Gemini behavior, research role, and learned rules.

## Current Project

Open `game-ai/index.html` in a browser to run the current game locally.

## Multi-Agent Flow

Use this default composition:

1. Claude clarifies and plans.
2. Gemini compares approaches when needed.
3. Codex implements and verifies.
4. Claude reviews the result.
5. Codex applies final fixes and updates docs.

The full workflow is documented in `docs/AI_COMPOSITION_WORKFLOW.md`.
